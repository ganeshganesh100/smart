# My Node.js Website

A lightweight, scalable web application built using Node.js and Express. This project serves as a foundation for building modern web applications with server-side rendering or API integrations.

---

## 🚀 Features

* **Fast & Minimalist:** Built on top of the Express web framework.
* **Environment Configuration:** Secure management of credentials using `dotenv`.
* **Static File Serving:** Built-in support for HTML, CSS, JavaScript, and images.
* **Modular Architecture:** Clean separation of routes, controllers, and views.

---

## 🛠️ Prerequisites

Before you begin, ensure you have the following installed on your local machine:

* **Node.js** (v18.0.0 or higher recommended)
* **npm** (Node Package Manager, comes bundled with Node)
